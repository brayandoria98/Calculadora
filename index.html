Calculadora: MIUI
¿Cómo usar las secciones especiales de la calculadora?
1. Para encontrar la raíz de algún numero basta con presionar el ícono de raíz "√" y poner el número + "," + el tipo de raíz que quieres buscar (Raíz cuadrada, raíz cúbica, etc) + ")". Por ejemplo para hallar la raíz cuadrada de 25 sería root(25, 2) = 5. Para hallar la raíz cúbica sería root(125, 3) = 5.
2. Para la potenciación se hace algo similar, se presiona el símbolo "^", luego se pone de primero el número que queremos elevar a una potencia + "," + la potencia + ")". Por ejemplo, queremos elevar 5 a la 2, sería pow(5,2) = 25.
3. El símbolo de "%" en este caso no significa porcentaje, significa "Resto" o "Remainder" en inglés, lo cuál es el residuo que queda en una división. Por ejemplo, 100%10 = 0 porque 10*10 = 100, entonces no queda ningún residuo.
